## Laravel-7 "Personal Area" Project

## System Requirements

To be able to run Laravel Boilerplate you have to meet the following requirements:

<ul>
<li>PHP &gt;= 7.3</li>
<li>Node.js &gt;= v14.x</li>
<li>Composer &gt;= 1.9.x</li>
</ul>
